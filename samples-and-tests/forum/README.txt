# This is a sample forum application built with the Play! framework
# ~ http://www.playframework.org
#
# Please use play-1.0-stable5

# To start :

Edit the conf/application.conf file, and update these configuration properties :

- forum.adminEmail -> Set the administrator email
- mail.smtp.* -> Configure correct STMP server (gmail sample is included)

type 'play run' in the application directory
